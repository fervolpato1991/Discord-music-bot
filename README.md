# 🎵 Discord Music Bot Camaleon DJ

</p>

Bot de música para Discord desarrollado en Python, con reproducción desde YouTube, sistema de cola, controles interactivos y panel dinámico tipo "Now Playing".

---

## 🚀 Features

- ▶️ Reproducción de música desde YouTube
- 📜 Soporte para playlists
- ⏭️ Cola automática de canciones
- 🎛️ Controles con botones (pause, resume, skip, stop)
- 📊 Barra de progreso en tiempo real
- 🧹 Limpieza automática de archivos temporales
- 👤 Identificación del usuario que pidió cada canción
- 🔄 Auto desconexión por inactividad
- 🧠 Panel dinámico tipo reproductor (Now Playing)

---

## 🖥️ Comandos

| Comando     | Descripción |
|------------|------------|
| `!play <url>` | Reproduce una canción o playlist |
| `!skip`       | Salta la canción actual |
| `!pause`      | Pausa la música |
| `!resume`     | Reanuda la música |
| `!stop`       | Detiene la reproducción |
| `!leave`      | Desconecta el bot |
